# Platform Resources Terraform Module

This module manages platform-wide Kubernetes resources that need to exist alongside the cluster, but are separate from the cluster provisioning itself.

## Usage

```bash
cd platform-resources
tofu init
tofu plan -var="letsencrypt_email=your-email@example.com"
tofu apply -var="letsencrypt_email=your-email@example.com"
```

## Resources

- **ClusterIssuer (letsencrypt-http01)**: Configures cert-manager to request TLS certificates from Let's Encrypt using the ACME HTTP-01 challenge via Cilium Gateway API

## Notes

- This module uses the kubeconfig generated by the workload-cluster module
- No changes to control plane nodes are required when applying this module
- The ClusterIssuer references the `cilium-gateway` Gateway in the `default` namespace
